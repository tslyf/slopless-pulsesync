# Slopless

Аддон для клиента PulseSync, защищающий вашу медиатеку от музыки, сгенерированной нейросетями.

## Что он делает?
1. Помечает ИИ-артистов бейджами в списках и на страницах профилей.
2. Умеет автоматически пропускать или ставить дизлайк сгенерированным трекам в момент их включения.

## Обратная связь и исходный код
